# ⛓️ Sliding Window vs Prefix Sums

| Technique        | Memory     | Speed  | Good for                     |
|----------------  |------------|--------|------------------------------|
| Sliding Window   | O(1) extra | O(n)   | Real-time updates, streams   |
| Prefix Sum (map) | O(n) extra | O(n)   | Many queries over same array |
